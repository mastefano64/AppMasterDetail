This is a master-deteil control for Angular (it is a preview)

You can find the project on GitHub at this link
MasterDetail is an Angular control that allows the display of data sources that are linked by a master-detail relationship. It offers various features, including pagination, list view, tabular view, insertion, editing, deletion, grouping/field (if tabular view is being used), creation of custom detail panels, etc... The various panels: "list", "tabular" or "custom"; however, they can be individually enabled/disabled.

The control supports 2 types of paging (internal/external): 1) the data is read all together and then paged through an internal navigation bar (example1a, example1b, example1c, example1d, example1f, example1g, example1h). 2) one piece of data is read at a time, passed to the MasterDetail control and paged through an external navigation bar. A "data source" external to the control is used, which is responsible for making the related rest calls (example2i, example2l, example2l ).

Il layout è gestito: 1) attraverso un oggetto di configurazione che viene poi passato al controllo MasterDetail e mediante il quale si attivano o si disabilitano le varie funzionalità. 2) attraverso dei template contenuti tra il tag di apertura e chiusura del controllo MasterDetail.
